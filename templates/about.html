My Programming Story

After learning the basics of JavaScript in my high school's "Intro to Programming" elective course and falling in love with programming during that year, I decided to major in Computer Science at University. At New York University, I learned Python, data structures and algorithms, and C++.

Outside of school, I also strive to learn other languages:

Back in 2019, I learned Java more in depth for a massive side project/business I was planning with a few friends who weren't programmers: the LW Network. It was this project that I got write code that had to be ran 24/7, and I would also learn how to interact with linux servers remotely via Putty.

In 2020, after COVID-19 arrived in New York and we entered self-isolation, I started learning how to create my own games. Now, I'm not a game developer, but working with Unity gave me the drive I needed to actually learn C# fundamentals. While I haven't finished a game, I am working on a simple 3D platformer that may be released eventually. I'm a bit overcritical of my own work, and actively seek out peer review to ensure what I choose to publish on Github isn't terrible.

I struggled a bit with C++, as Python was the first language I would consider myself "fluent" in. That being said, I've come to fall in love with C++ after working on a few side projects. I'd consider my first proper program in C++ to be my attempt at recreating the "Petals Around the Rose" game on my GitHub, which I haven't had time to go back and edit as I'm working on a Blockchain implementation based off of Free Code Academy's Python Blockchain (which I transliterated into C++ and got it to work!). I plan on eventually branching that out into a small cryptocurrency to use amongst friends as a form of "Monopoly Play Money". Y'know, to make Poker Nights a bit more intense.

And finally, in 2021, I decided to start building a website with Flask in order to better understand HTMl5 and the Flask framework. Now this site is effectively static, but it'll eventually have some dynamic content as a sort of puzzle game inspired by Cicada-3301's puzzles. Currently, it's main purpose is an online résumé, a place to showcase my work, and a medium for people to recommend me with.

It also serves as a way for old friends to find my up-to-date information in the event they wish to reconnect for whatever reason - don't be afraid to reach out!
