My Programming Story

My first "Hello World"

Everybody has a beginning. My beginning was June 5th, 2000. It was a Monday...

Nah I'm just messing with you.

The first program I ever wrote was back in 2013. I was thirteen year old at the time, and I was playing on a Minecraft multiplayer server with a few friends. Neither one of us were hosting the server at the time, and it was a team-based server where you survive with friends while sabotaging other players.
We weren't doing so hot as we we were being targetted by the other players due to our inexperience in such a setting. So I had a brilliant idea:

"If I were to somehow gain access to administrative privileges, we could rule the server!"

The server we were on was a Bukkit server. Bukkit is an open-source framework meant to extend the default Minecraft Server's functionality. It contains a built in API which allows anyone to write Java code that the server can execute from in-game commands. While it's designed to be secure and safe, there are no protections in place against plugins running detrimental code, such as a player executing an in-game command that causes the server to promote said player to have administrative status.

I swiftly opened up a Google Chrome window and googled "How to code a force-op plugin with Bukkit", and to my 14 year old surprise I found what I was looking for: a 20 minute tutorial video that guides you on interfacing with the Bukkit API in the Eclipse IDE, and how to actually set up such a plugin.

Despite this being JavaScript, the code I wrote pretty much looked like this.

I remember spending at least an hour following that guide, not knowing what I was doing or what anything meant. But I was determined to create this plugin, and I had all of the freetime in the world since it was a weekend.

I eventually got the program to compile, and through some social engineering I managed to get the server owner to install the plugin. I ran the custom command I wrote that would give myself administrative privileges, and to me shock and surprise it worked.

Looking back at it, 8 years later as a 21 year old. What I did was obviously very wrong. Instead of writing "System.out.print("Hello World")", my first program was a a backdoor that granted me administrative powers. Thankfully, the only malicious act I did was just spawn in rare items and built ourselves a base beyond the server's set world border so we couldn't get raided.

The reason I share the story, despite it's origins as a 14 year old's "lust for power", is because it opened my eyes to what you could do with programming. While I felt like a "Hacker Man" at the time, it was my first experience with building and deploying a piece of software with a real-world purpose and use case. It also taught me that anyone can program - you didn't have to be a college student or child prodigy to understand how to code.

Also as if it isn't obvious: I never coded another backdoor like that ever again.
